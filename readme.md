# murmurhash

> A ES5 wrapper for the MurmurHash function

## Install ##

    npm install --save @4keys/murmurhash

or

    yarn add murmurhash


## Usage ##

```js
var murmurhash = require('murmurhash');
// or
import murmurhash from 'murmurhash';

murmurhash('Any String')
//=> 256561456446
```
